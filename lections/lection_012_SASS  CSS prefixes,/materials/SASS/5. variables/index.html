Переменные

Переменные - это возможность при помощи которой можно использовать цвета, дефолтные значения, единицы, в любом месте вашего Sass файла.

Изменив значение переменной в месте ее создания - значение поменяется во всех местах где данная переменная была использована.

Переменная определяется так: $название: значение.
